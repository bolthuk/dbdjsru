# bot.onRoleDelete

```javascript
bot.roleDeleteCommand({
        channel: "channel ID",
        code: `your code`
        })
Bot.onRoleDelete()
```

```javascript
bot.roleDeleteCommand({ 
        channel: "772414449839636490", 
        code: `Роль удалена:
        Старая роль: $oldRole[name]`
        })

```

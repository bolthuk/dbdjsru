# bot.onReady

```javascript
bot.readyCommand({
    channel: "channel ID",
    code: `your code`
})
```

Using the callback

```javascript
bot.readyCommand({
    channel: "772414449839636490",
    code: `Я онлайн и готов к работе`
}) //bot.onReady() не нужен
```
